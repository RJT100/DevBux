DevBux API Usage

Balance API

The Balance API allows you to retrieve the balance of a user's DevBux.

To use the Balance API:

  • Send a GET/HTTP request to balapi.php with the following parameters:
    • username: The username of the user whose balance you want to retrieve.
  • Example: balapi.php?username=user123

Send API

The Send API allows you to transfer DevBux from one user to another.

To use the Send API:

  • Send a GET/HTTP request to sendapi.php with the following parameters:
    • username: Your username.
    • password: Your password.
    • sendto: The username of the recipient.
    • amount: The amount of DevBux to send.
  • Example: sendapi.php?username=user123&password=password123&sendto=recipient456&amount=10

Make sure to replace the placeholder values (user123, password123, recipient456) with actual usernames and passwords.
